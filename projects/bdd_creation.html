Création d'une Base De Donnée

Description du projet

Il s'agit de réaliser une base de donnée à partir d'un cahier des charges et d'un fichier CSV représentant toutes les tables unifiées.
Ainsi j'ai réalisé un diagramme de classe UML afin de pouvoir représenter la base de donnée à partir de ces données.
Puis grâce au diagramme j'ai pu implémenter la base de donnée.
