Convertisseur de PDF

Description du projet

Projet d'équipe permettant de convertir des fichiers textes en fichier pdf, pour cela nous avons:

  • Créer une template HTML qui montre à quoi ressemblera un fichier PDF converti
  • Réaliser différents scripts permettant de séparer les différents morceaux d'un fichier texte et mettant ces parties dans des fichiers .dat
  • Faire un script permettant de prendre le contenu des fichiers .dat et de les transformer en un fichier .html
  • Convertir le fichier HTML en un fichier PDF
  • Puis faire un script main permettant de réaliser ces différentes tâches dans l'ordre
