Puissance4

Description du projet

Ce projet conciste en l'implémentation d'un puissance4 en C qui s'éxécute dans le terminal.
Pour cela j'ai réalisé:

  • Une maquette détaillant les différentes fonctions
  • Un document détaillant l'apparence graphique qu'aura le puissance4 dans le terminal
  • Le code C commenté du puissance4 (joueur VS joueur)
  • Réaliser les tests
Puis pour créer une version contre l'ordinateur:
  • Analyser le pourcentage de victoire d'une stratégie (alétoire, privilégier le milieu...) par rapport à d'autres
  • Etablir une stratégie globale pour l'ordinateur
  • Implémenter l'ordinateur en essayant d'optimiser le temps d'éxecution de ce dernier
